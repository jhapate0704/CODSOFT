## 📂 Projects

### 1. Standard Calculator 🧮

A modern, responsive calculator that handles basic arithmetic operations with a clean, animated interface and full keyboard support.

<img width="1919" alt="Calculator Project Screenshot" src="https://github.com/user-attachments/assets/af6c2436-f7da-4394-a693-19ce59ba6725" />


#### Task Objective
The goal was to build a functional calculator to demonstrate core JavaScript logic, DOM manipulation skills, and the ability to handle user input through both clicks and keyboard events.

#### Features
- **Core Operations:** `+`, `-`, `×`, `÷`, `%`, decimal input, and backspace.
- **Responsive Design:** Flawless performance on both desktop and mobile devices.
- **Keyboard Support:** Enables faster calculations using the physical keyboard.
- **Modern UI/UX:** A sleek design with smooth animations and visual feedback.

#### Technical Highlights
- Implemented robust event listeners for all buttons and keyboard inputs.
- Developed the core calculation logic using JavaScript functions and string manipulation.
- Styled with modern CSS, including gradients and transitions for an improved user experience.

### 2. Blog Landing Page 📝

A beautifully designed, fully responsive blog landing page. It features a modern layout with sidebar widgets and mobile-friendly navigation, perfect for any content-focused website.

<img width="1910" alt="Landing Page Project Screenshot" src="https://github.com/user-attachments/assets/cbc4d079-3c0e-4881-bfe5-efc1cac02835" />


#### Task Objective
This task was designed to assess my ability to create a static, responsive web page from scratch, focusing on semantic HTML structure and advanced CSS layout techniques.

#### Features
- **Fully Responsive:** Adapts perfectly to all screen sizes using media queries.
- **Clean Typography:** Uses Google Fonts for a professional and readable look.
- **Functional Sidebar:** Includes widgets for search, recent posts, and categories.
- **Semantic HTML:** Optimized for SEO and accessibility with tags like `<main>`, `<aside>`, and `<nav>`.

#### Technical Highlights
- Utilized CSS Flexbox and Grid to create a complex, multi-column responsive layout.
- Implemented a mobile-first design approach.
- Ensured semantic structure for better accessibility and search engine visibility.

### 3. Personal Portfolio Website 👨‍💻

A dynamic personal portfolio to showcase my skills, feature my best projects, and provide a way for visitors to connect with me. Includes a theme toggle for user preference.

<img width="1900" alt="Portfolio Project Screenshot" src="https://github.com/user-attachments/assets/1db39b42-b567-4ab6-8343-07108f057383" />

**[➡️ Live Demo](https://nirajprofile.netlify.app/)** | 

#### Task Objective
To create a comprehensive personal portfolio that not only lists my skills and projects but also demonstrates my design sense and JavaScript interactivity, such as a theme switcher.

#### Features
- **Multi-Theme:** Switch between light and dark modes, with the user's preference saved in local storage.
- **Interactive Sections:** Clearly defined areas for skills, projects, and contact info.
- **Responsive Layout:** Looks great on desktops, tablets, and mobile phones.
- **Contact Form:** A functional form for visitors to send messages (front-end only).

#### Technical Highlights
- Wrote JavaScript to toggle CSS classes on the body element for theme switching.
- Used `localStorage` to persist the chosen theme across browser sessions.
- Structured the site with reusable components and a clean, easy-to-update layout.

## 🚀 How to Run Locally

To run any of these projects on your local machine, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/jhapate0704/Your-Repository-Name.git](https://github.com/jhapate0704/Your-Repository-Name.git)
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd Your-Repository-Name/project-folder-name
    ```
    (e.g., `cd Your-Repository-Name/calculator-folder-name`)

3.  **Open the `index.html` file in your favorite web browser.**
